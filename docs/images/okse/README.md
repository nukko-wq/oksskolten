# Okse Logo Assets

Bull icon inspired by Oksskolten, a mountain known as "the bull's head".

| File | Format | Description |
|---|---|---|
| `okse-original.png` | PNG | Original logo (white background) |
| `okse.svg` | SVG | 2-color vector (blue-gray shadow + dark outline) |
| `okse.png` | PNG | 2-color raster (rendered from SVG) |
| `okse-mono.svg` | SVG | Monochrome silhouette vector |
| `okse-mono.png` | PNG | Monochrome silhouette raster (rendered from SVG) |
| `okse-transparent.png` | PNG | 2-color with transparent background |
